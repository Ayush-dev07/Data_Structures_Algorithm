## 💡 Problem Solving Strategy

For every problem, I try to follow the same workflow:

1. Understand the problem thoroughly.
2. Think of a brute-force approach.
3. Analyze its complexity.
4. Optimize the solution.
5. Implement clean, readable code.
6. Document the approach and learnings.
